# Cricket-analysis-dashbord


# Project overview
The Cricket Analysis Dashboard is an interactive Excel-based data analytics project designed to analyze and visualize cricket player performance metrics. The dashboard provides meaningful insights through dynamic charts, KPIs, and pivot-table-driven reports.

# Dataset
<a href="https://github.com/indhuindhu6302-bit/cricket-analysis-dashbord/blob/main/cricket%20analysis%20dashboard.xlsx">Data</a> 


# QUASTIONS(KPI)
-Who is the top run scorer in the dataset?
-Which player has the highest batting average?
-Which player has the highest strike rate?
-Who has taken the most wickets?
-Which country has scored the highest total runs?
-Which country has the highest number of wickets?
-What is the total number of matches played by all players?
-Who has the highest individual score?
-Which player has the best economy rate?
-What is the average strike rate across all players?
